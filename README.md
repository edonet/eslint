# @ainc/eslint-config-recommended
Let's do something nice with @ainc/eslint-config-recommended!

## Install
``` shell
$ yarn add --dev @ainc/eslint-config-recommended
```

## Usage
``` json
{
  "extends":["@ainc/recommended"]
}
```
``` json
{
  "scripts": {
    "lint": "eslint src --ext .js,.jsx,.ts,.tsx,.html",
    "lint:fix": "eslint src --ext .js,.jsx,.ts,.tsx,.html --fix"
  }
}
```
